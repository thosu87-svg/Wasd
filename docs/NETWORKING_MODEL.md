# Networking Model

Der Server bleibt autoritativ.

## Flüsse
- Client -> Input Packets -> Server
- Server -> State Updates -> Client
- GM Editor -> Command Packets -> Server Validation -> Broadcast

## Synchronisierte Zustände
- Spielerpositionen
- NPC-Sichtzustände
- Chunkaktivierung
- Kampfergebnisse
- Eventsignale
