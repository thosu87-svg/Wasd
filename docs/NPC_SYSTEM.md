# NPC System

## Fünf Einflussfaktoren
- DNA / Traits
- Familie / Genealogie
- Kultur / Herkunft
- Religion / Weltbild
- Erfahrung / Memory

## Memory-Ebenen
- Local memory
- Social memory
- Faction memory
- Historical memory
- Oracle memory

## Verhalten
NPCs können:
- arbeiten
- handeln
- Gerüchte verbreiten
- Familien gründen
- Politik betreiben
- Kriege beginnen oder vermeiden
