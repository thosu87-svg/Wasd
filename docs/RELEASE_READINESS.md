# Release Readiness

## Vor Release prüfen
- Health endpoint antwortet
- WebSocket verbindet
- Client rendert
- kritische Panels öffnen
- Persistenzpfade gesetzt
- Secrets extern konfiguriert
- GM-Rechte geprüft
- Rekonstruktionspack aktuell
